# WaifuGUI
A waifu-converter-cpp interface.

This program is for Windows only.
You can download it from the Releases tab.

# Build
1- Clone it

2- In the bin folder, simply add opencv_world330.dll(You can get it from the Releases tab)

3- Install Python libraries wx, colorama, cx_Freeze

4- Run make.bat

5- Copy bin and models_rgb folders into build/exe.winXX-3.X.

6- Enjoy !
